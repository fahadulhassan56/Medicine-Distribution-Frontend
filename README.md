## Medicine-Distribution

## Overview
A frontend application for managing medicine distribution workflows including inventory, suppliers, and stock tracking UI.

---

## Features
- Dashboard UI for medicine tracking
- Supplier management screens
- Inventory visualization
- Responsive UI design

---

## Tech Stack
- React / Next.js
- Tailwind CSS
- API Integration

---

## Project Status
Frontend-only application (Backend API Integration Ready)
